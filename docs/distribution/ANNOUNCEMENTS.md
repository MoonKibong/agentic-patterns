## Longer Post

```markdown
I started Agentic Patterns: A Pattern Language for Human-Guided AI Work.

The motivation is practical. Agentic coding has made some reusable workflow shapes visible:
context packets, role contracts, tool-use boundaries, review loops, durable state, and human approval
gates. The same shapes also show up in image generation, writing, design, video/media, and research.

The project is not a prompt dump. It is a catalog of reusable patterns with:

- applicability
- forces and tradeoffs
- procedures
- examples
- failure modes
- evidence hygiene

The repo is seed-stage and intentionally uses synthetic/sanitized examples so private workflows do
not leak.

Repo: <repo-url>
```
