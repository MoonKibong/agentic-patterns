# Discoverability Plan

This is the maintainer checklist for making Agentic Patterns findable after public release.

## Current Positioning

Agentic Patterns is a pattern language for human-guided AI work. It is not a prompt dump and not a
vendor-specific agent framework. It names reusable context, prompt, harness, orchestration, tool,
memory, and evaluation patterns across coding and creative workflows.

## Launch Checklist

1. Finalize license.
2. Publish the repo with seed-stage language.
3. Create a GitHub release from `CHANGELOG.md`.
4. Add GitHub topics.
5. Share a short announcement focused on the problem, not hype.
6. Invite contributions for field notes, translations, and new patterns.
7. Track feedback about confusing taxonomy or missing domains.
