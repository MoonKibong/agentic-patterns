# Design Generation

Design generation includes product design, interface concepts, design-to-code, visual systems,
layout exploration, and asset handoff.

## Common Forces

- Design outputs must balance aesthetics, usability, brand, constraints, and implementation cost.
- Agents may generate beautiful artifacts that cannot be implemented or maintained.
- Design-to-code requires strict handoff between visual intent and technical execution.
- Human review should happen before expensive downstream implementation.
