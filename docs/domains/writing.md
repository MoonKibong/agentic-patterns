# Writing And Literature

Writing workflows use agentic patterns when the work spans briefs, outlines, drafts, critique,
revision, continuity, citations, or editorial voice.

## Common Forces

- Single-pass writing can sound polished while missing intent.
- Long-form work needs memory for characters, claims, structure, and voice.
- Critique helps only when grounded in an explicit rubric.
- Human taste, originality, and ethical constraints remain central.
