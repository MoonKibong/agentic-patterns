# Image Generation

Image generation benefits from agentic patterns when a user needs more than one prompt: references,
style continuity, variations, critique, selection, and handoff into downstream design or media work.

## Common Forces

- A loose prompt creates visually plausible but misaligned images.
- References and style words can conflict.
- Human taste is central and should not be hidden behind automatic scoring.
- Iteration can be expensive or drift away from the original intent.
