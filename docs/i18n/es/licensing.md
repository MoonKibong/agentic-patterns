# Licencia

Este proyecto usa una licencia dividida.

## Decisión Actual

Usa un modelo de licencia dual:

- Catálogo de patrones, documentación, prompts, notas de campo: **Creative Commons Attribution 4.0 International (CC BY 4.0)**.
- Código, esquemas, artefactos de software aprobados: **MIT**.

## Motivo

El catálogo es trabajo de conocimiento. CC BY 4.0 fomenta la reutilización, adaptación y
traducción manteniendo la atribución.

Para artefactos de software se usa MIT. MIT es simple, ampliamente comprendido y fácil de
entender para los contribuidores.

## Alternativa

Si el lenguaje explícito de patentes es importante, se puede usar Apache-2.0 para los
artefactos de software.

## Archivos del Repositorio

- `LICENSE`: resumen de la licencia dividida y términos de contribución.
- `LICENSE-MIT`: texto de la licencia MIT que aplica a los artefactos de software.
- URL oficial de CC BY 4.0: https://creativecommons.org/licenses/by/4.0/

Antes de la publicación, los contribuidores deben entender que los patrones y notas de campo
enviados se aceptan bajo la licencia del repositorio.
