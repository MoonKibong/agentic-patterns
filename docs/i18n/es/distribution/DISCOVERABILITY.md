# Plan de Descubribilidad

Este es el checklist del mantenedor para hacer que Agentic Patterns sea encontrable después
del lanzamiento público.

## Posicionamiento Actual

Agentic Patterns es un lenguaje de patrones para el trabajo con IA guiado por humanos. No
es un volcado de prompts ni un framework de agentes específico de un proveedor. Nombra
patrones reutilizables de contexto, prompt, harness, orquestación, herramientas, memoria y
evaluación en flujos de trabajo de coding y creativos.

## Checklist de Lanzamiento

1. Finalizar la licencia.
2. Publicar el repositorio con lenguaje de etapa semilla.
3. Crear un release de GitHub desde `CHANGELOG.md`.
4. Añadir temas de GitHub.
5. Compartir un anuncio corto enfocado en el problema, no en el hype.
6. Invitar contribuciones de notas de campo, traducciones y nuevos patrones.
7. Rastrear feedback sobre taxonomía confusa o dominios faltantes.
