# Patrones de Contexto

Los patrones de contexto definen cómo empaquetar, reducir, preservar y transferir información
para el trabajo agéntico.

Usa patrones de contexto cuando un agente necesita suficiente información para actuar
correctamente, pero un volcado de archivos, historial, logs o chat ocultaría la señal importante.

## Patrones Actuales

- [Paquete de Contexto (Context Packet)](../../../../patterns/context/context-packet.md): proporciona contexto
  delimitado y orientado a la tarea antes de planificar, ejecutar, revisar o transferir.
- [Paquete de Brief Creativo (Creative Brief Packet)](../../../../patterns/context/creative-brief-packet.md):
  empaqueta intención creativa, restricciones, referencias, entregables y criterios de evaluación
  para trabajo generativo.

## Ideas de Patrones

- Handoff Summary.
- Constraint Header.
- Evidence Bundle.
- Resume Capsule.
