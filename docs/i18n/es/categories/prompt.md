# Patrones de Prompt

Los patrones de prompt definen estructuras de instrucción reutilizables para roles, tareas,
restricciones y salidas esperadas.

Usa patrones de prompt cuando el problema principal no es código faltante ni herramientas
faltantes, sino comportamiento poco claro: qué debe optimizar el agente, qué puede decidir
y qué debe devolver.

## Patrones Actuales

- [Contrato de Rol (Role Contract)](../../../../patterns/prompt/role-contract.md): define un rol de agente
  mediante autoridad, responsabilidades, límites, salidas y reglas de escalada.

## Ideas de Patrones

- Reviewer Gate.
- Output Contract.
- Assumption Ledger.
- Decision Escalation.
