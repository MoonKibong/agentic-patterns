# Patrones de Memoria

Los patrones de memoria definen cómo el conocimiento útil sobrevive más allá de un único
turno de agente, sesión de chat o flujo de trabajo.

Usa patrones de memoria cuando el trabajo repetido depende de lecciones, decisiones,
restricciones o estado que no deben redescubrirse cada vez.
