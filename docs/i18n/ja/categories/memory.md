# メモリパターン

Memory pattern は有用な知識が単一のエージェントターン、チャットセッション、
ワークフローを超えて残る方法を定義します。

反復する作業がレッスン、決定、制約、状態に依存し、毎回再発見すべきでない場合に使用します。
