# 評価パターン

Evaluation pattern は人間とエージェントがパターン、計画、ワークフロー、出力が
適切かつ効果的かどうかを判断する方法を定義します。

中心的な問いが「エージェントは何をすべきか?」ではなく「このアプローチが適切で
機能したかどうかをどうやって知るか?」である場合に使用します。

## 現在のパターン

現在公開されている評価パターンはありません。非公開の評価パターンは公開ウィキの外で管理されています。

## パターンアイデア

- Evidence Strength.
- Pattern Maturity.
- Counterexample Review.
- Outcome Trace.
