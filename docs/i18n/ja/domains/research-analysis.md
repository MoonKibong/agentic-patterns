# 調査と分析

Research と analysis のワークフローはコンテキスト、ソース管理、引用規律、批評、
および結論に対する人間の判断を必要とします。

## 共通の Forces

- エージェントは流暢だが根拠のない主張を生成することがあります。
- 長いリサーチタスクには永続的なソースリストと決定記録が必要です。
- ツールの出力は信頼できなかったり古くなっていたりすることがあります。
- 人間は洗練された要約だけでなく、透明な出典が必要です。
