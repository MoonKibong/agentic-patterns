# 執筆と文学生成

Writing ワークフローは、作業がブリーフ、アウトライン、ドラフト、批評、改訂、
一貫性、引用、編集上の声をまたぐ場合に agentic pattern を使用します。

## 共通の Forces

- シングルパスの執筆は洗練されて見えながら意図を見落とすことがあります。
- 長編作業にはキャラクター、主張、構造、声のためのメモリが必要です。
- 批評は明示的なルーブリックに基づいている場合のみ有用です。
- 人間のセンス、独自性、倫理的制約が中心です。
