# 発見可能性プラン

このドキュメントは Agentic Patterns を公開した後に人々が見つけられるようにするための
メンテナーチェックリストです。

## 現在のポジショニング

Agentic Patterns は human-guided AI work のためのパターン言語です。プロンプトダンプでも
ベンダー固有のエージェントフレームワークでもありません。コーディングとクリエイティブな
ワークフロー全体にわたるコンテキスト、プロンプト、ハーネス、オーケストレーション、
ツール、メモリ、評価パターンに名前をつけます。

## 公開チェックリスト

1. ライセンスを確定する。
2. シードステージの言語でリポジトリを公開する。
3. `CHANGELOG.md` から GitHub リリースを作成する。
4. GitHub トピックを追加する。
5. 誇大宣伝ではなく問題中心の短い告知を共有する。
6. フィールドノート、翻訳、新しいパターンへのコントリビューションを招待する。
7. 分類体系の混乱や不足しているドメインに関するフィードバックを追跡する。
