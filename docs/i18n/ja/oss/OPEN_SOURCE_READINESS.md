# Open Source Readiness（オープンソース準備）

このドキュメントは Agentic Patterns の公開リリースにおける推奨デフォルトを記録します。

## Task Evaluation

Verdict: `REVISE`（ライセンスが確定し、翻訳済みクイックスタートと完全な wiki ミラーの
人間によるレビューパスが完了するまで）。

Scope: repository docs、pattern catalog、GitHub template、i18n、contribution workflow、
public positioning。

プライマリリスク: このプロジェクトは実際の agentic ワークフローから着想を得ているため、
公開ドキュメントは独自プロンプト、非公開ハーネスの詳細、顧客データ、プライベートリポジトリ名、
内部ワークフロートランスクリプトを公開してはなりません。

## ポジショニング

Agentic Patterns は human-guided AI work のためのパターン言語として提示します。
プロンプト集ではなく、ベンダー固有のエージェントフレームワークでもありません。
Coding は最初の成熟したドメインですが、カタログは画像生成、執筆、デザイン、
video/media、リサーチ、分析もカバーします。

## ライセンス

現在の決定:

- ドキュメント、パターンカタログ、プロンプト、フィールドノート: CC BY 4.0。
- コード、スキーマ、承認されたソフトウェアアーティファクト: MIT。

## リリースステージ

最初の公開リリースを `v0.1.0` のシードステージ言語で公開します。
ドラフトパターンを実証済みのベストプラクティスとして提示しないでください。

## 証拠とプライバシーポリシー

Synthetic、sanitized、または明示的に公開された例を要求します。
非公開プロンプト、顧客データ、非公開 URL、独自ワークフロー詳細の偶発的な漏洩は
セキュリティセンシティブとして扱います。

## CI

この Markdown ファースト公開リポジトリの CI は、公開バリデーターが承認されたら
markdown のリントとリンクチェックから始めることができます。

## コントリビューション

PR には以下を含める必要があります:

- ユーザー向けドキュメントまたはパターン変更の概要
- 検証コマンド
- 証拠・プライバシーへの影響
- i18n への影響
- 関連 issue へのリンク

## 公開チェックリスト

1. ライセンスファイルを確定する。
2. `README.md`、`CONTRIBUTING.md`、`SECURITY.md`、`CODE_OF_CONDUCT.md`、`CHANGELOG.md`、`RELEASE.md` を確認する。
3. issue と PR テンプレートの存在を確認する。
4. 承認された公開レビューと検証チェックを実行する。
5. `docs/i18n/` クイックスタートと wiki ミラーをレビューする。
6. 非公開の用語、非公開 URL、認証情報、内部リポジトリ名を検索する。
7. シードステージ言語で GitHub リリースを作成する。

## 決定チェックリスト

| 項目 | 推奨 | ステータス |
| --- | --- | --- |
| 製品の約束 | Human-guided AI work のためのパターン言語 | 文書化済み |
| ライセンス | CC BY 4.0 ドキュメント + MIT コード | 存在する |
| README | スコープ、カタログ、検証、証拠衛生 | 存在する |
| Changelog | Keep a Changelog スタイル（`Unreleased` あり） | 存在する |
| Contributing | パターン、証拠、i18n の期待値 | 存在する |
| Security | 非公開漏洩と安全でない自動化の報告 | 存在する |
| Code of conduct | 軽量なメンテナー強制ポリシー | 存在する |
| Issue templates | パターン提案あり; さらなるテンプレートを推奨 | 部分的 |
| PR template | 概要、検証、i18n、プライバシーへの影響 | 存在する |
| CI | 公開 markdown/リンクチェック | 未実装 |
| リリースプロセス | シード `v0.1.0` チェックリスト | 存在する |

## 受け入れ基準

- 公開ドキュメントがプロジェクトの内容と対象外を説明している。
- メンテナーがリリースとロールバックの指示を持っている。
- コントリビューターがセットアップ、検証、パターン、プライバシー、i18n の期待値を持っている。
- ローカライズされたクイックスタートが存在し、英語ドキュメントが真実のソースであることを記載している。
- パターンを理解または貢献するのに独自ソリューションの詳細が不要である。
