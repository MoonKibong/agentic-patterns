# Licensing（ライセンス）

このプロジェクトはスプリットライセンスを使用します。

## 現在の決定

デュアルライセンスモデルを使用します:

- パターンカタログ、ドキュメント、プロンプト、フィールドノート: **Creative Commons Attribution 4.0 International (CC BY 4.0)**。
- コード、スキーマ、承認されたソフトウェアアーティファクト: **MIT**。

## なぜこれが適切か

カタログは知識作業です。CC BY 4.0 は attribution を保持しながら再利用、改変、翻訳、引用を奨励します。

ソフトウェアアーティファクトには MIT を使用します。MIT はシンプルで、広く理解されており、
許容的で、コントリビューターが理解しやすいためです。

## 代替案

特許条項が重要な場合はソフトウェアに Apache-2.0 を使用できます。

## Repository Files

- `LICENSE`: スプリットライセンスの概要とコントリビューション条件。
- `LICENSE-MIT`: ソフトウェアアーティファクトに適用される MIT ライセンステキスト。
- CC BY 4.0 の公式 URL: https://creativecommons.org/licenses/by/4.0/

公開前に、コントリビューターが提出したパターンとフィールドノートがリポジトリライセンス
のもとで受け入れられることを理解していることを確認してください。
