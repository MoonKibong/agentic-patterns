# Patterns d'évaluation

Les patterns d'évaluation définissent comment les humains et les agents jugent si un pattern, un
plan, un workflow ou une sortie est approprié et efficace.

Utilisez les patterns d'évaluation lorsque la question centrale n'est pas « que doit faire l'agent
? » mais « comment savons-nous que cette approche convient ou a fonctionné ? »

## Patterns actuels

Aucun pattern d'évaluation public n'est actuellement répertorié. Les patterns d'évaluation privés
sont maintenus en dehors du wiki public.

## Idées de patterns

- Evidence Strength.
- Pattern Maturity.
- Counterexample Review.
- Outcome Trace.
