# Patterns de mémoire

Les patterns de mémoire définissent comment les connaissances utiles survivent au-delà d'un seul
tour d'agent, d'une session de chat ou d'un workflow.

Utilisez les patterns de mémoire lorsque le travail répété dépend de leçons, de décisions, de
contraintes ou d'états qui ne doivent pas être redécouverts à chaque fois.
