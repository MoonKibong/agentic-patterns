# Wiki Agentic Patterns

Bienvenue sur le wiki en français d'Agentic Patterns.

Naviguez à partir de votre question : suivez les pages de catégories et de domaines, puis ouvrez
les entrées de patterns individuels pour obtenir des détails opérationnels.
Le contenu anglais fait foi ; ce document est le miroir français du wiki public actuel.

## Pour les agents

- Lisez `AGENTS.md` à la racine du dépôt avant de modifier quoi que ce soit.
- Respectez la frontière public/privé approuvée par les mainteneurs avant de modifier les docs publics.
- Exécutez les procédures de validation du dépôt après avoir modifié des fichiers de patterns.

## Pour les contributeurs

- Commencez par le [Guide de contribution](CONTRIBUTING.md).
- Utilisez le [Modèle de pattern](../../../patterns/_template.md) pour les nouveaux patterns.
- Utilisez le [Modèle de note de terrain](../../../field-notes/_template.md) pour les observations précoces.
- Vérifiez les [Licences](licensing.md) avant les décisions de publication publique.
- Vérifiez l'[Hygiène des preuves](evidence-hygiene.md) avant d'ajouter des exemples ou des notes de terrain.
