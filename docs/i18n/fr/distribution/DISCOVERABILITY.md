# Plan de découvrabilité

Voici la liste de contrôle du mainteneur pour rendre Agentic Patterns trouvable après la
publication publique.

## Positionnement actuel

Agentic Patterns est un langage de patterns pour le travail IA guidé par des humains. Ce n'est pas
un déversement de prompts et ce n'est pas un framework d'agents spécifique à un fournisseur. Il
nomme des patterns de contexte, de prompt, de harnais, d'orchestration, d'outils, de mémoire et
d'évaluation réutilisables à travers les workflows de coding et créatifs.

## Liste de contrôle de lancement

1. Finaliser la licence.
2. Publier le dépôt avec le langage en phase initiale.
3. Créer une release GitHub à partir de `CHANGELOG.md`.
4. Ajouter les topics GitHub.
5. Partager une annonce courte centrée sur le problème, pas le battage médiatique.
6. Inviter des contributions pour les notes de terrain, les traductions et les nouveaux patterns.
7. Suivre les retours sur la taxonomie confuse ou les domaines manquants.
