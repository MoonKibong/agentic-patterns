# Agentic Patterns 快速开始

Agentic Patterns 是一种用于人类引导 AI 工作的模式语言。

它收集可复用的提示、上下文包、工作流 harness、角色契约、工具使用协议、记忆
方法和评估循环，用于重复性的 AI 工作流。

## 范围

项目从 agentic coding 开始，但也覆盖：

- 图像生成
- 写作和文学生成
- 设计生成
- 视频和媒体
- 研究和分析

## 核心分类

- `prompt`: 可复用的指令和角色结构
- `context`: 任务包、创意简报、记忆摘要
- `harness`: plan/execute/review 与 generate/critique/refine 循环
- `orchestration`: 多代理和多会话协调
- `tool`: shell、browser、MCP、API 和外部工具边界
- `memory`: 持久状态和可复用知识
- `evaluation`: 模式适配检查、评分标准和评审方法

## Evidence Hygiene

示例应为 synthetic、sanitized 或 public。不要发布私有提示、私有 harness 细节、客户
数据、凭据、private URL 或内部仓库名称。
