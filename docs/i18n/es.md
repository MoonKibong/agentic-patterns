# Inicio rapido de Agentic Patterns

Agentic Patterns es un lenguaje de patrones para trabajo con IA guiado por humanos.

Reune prompts reutilizables, paquetes de contexto, harnesses, contratos de rol, protocolos de uso
de herramientas, memoria y bucles de evaluacion para flujos de IA repetidos.

## Alcance

El proyecto empezo con agentic coding, pero tambien cubre:

- generacion de imagenes
- escritura y literatura
- generacion de diseno
- video y medios
- investigacion y analisis

## Categorias principales

- `prompt`: instrucciones reutilizables y estructuras de rol
- `context`: paquetes de tarea, briefs creativos, resumenes de memoria
- `harness`: bucles plan/execute/review y generate/critique/refine
- `orchestration`: coordinacion multiagente y multisesion
- `tool`: limites para shell, browser, MCP, API y herramientas externas
- `memory`: estado durable y conocimiento reutilizable
- `evaluation`: chequeos de ajuste, rubricas y metodos de revision

## Evidence Hygiene

Usa ejemplos sinteticos, sanitizados o publicos. No publiques prompts privados, detalles privados de
harnesses, datos de clientes, credenciales, private URLs ni nombres de repositorios internos.
