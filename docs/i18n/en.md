# Agentic Patterns Quick Start

Agentic Patterns is a pattern language for human-guided AI work.

It collects reusable prompts, context packets, harnesses, role contracts, tool-use protocols,
memory practices, and evaluation loops for repeated AI workflows.

## Scope

The project started from agentic coding, but it also covers:

- image generation
- writing and literature
- design generation
- video and media
- research and analysis

## Core Categories

- `prompt`: reusable instructions and role structures
- `context`: task packets, creative briefs, memory summaries
- `harness`: plan/execute/review and generate/critique/refine loops
- `orchestration`: multi-agent and multi-session coordination
- `tool`: shell, browser, MCP, API, and external-tool boundaries
- `memory`: durable state and reusable knowledge
- `evaluation`: pattern fit checks, rubrics, and review methods

## Evidence Hygiene

Use synthetic, sanitized, or public examples. Do not publish private prompts, private harness
details, customer data, credentials, private URLs, or internal repository names.
