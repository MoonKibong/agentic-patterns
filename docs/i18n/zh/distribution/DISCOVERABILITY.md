# 可发现性计划

本文档是 Agentic Patterns 公开发布后提升可发现性的 maintainer checklist。

## 当前定位

Agentic Patterns 是一种用于 human-guided AI work 的 pattern language。它既不是 prompt dump，也不是 vendor-specific agent framework。它在 coding 和 creative workflow 中为 context、prompt、harness、orchestration、tool、memory 和 evaluation pattern 命名。

## 发布检查清单

1. 确认 license。
2. 以 seed-stage language 将 repo 设为公开。
3. 基于 `CHANGELOG.md` 创建 GitHub release。
4. 添加 GitHub topics。
5. 撰写以实际问题为重心的公告，避免夸大宣传。
6. 邀请 field note、translation 和 new pattern 贡献。
7. 跟踪关于分类混乱或缺失领域的反馈。
