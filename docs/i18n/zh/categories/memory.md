# 记忆模式

Memory pattern 定义有用知识在单次 agent turn、chat session 或 workflow 结束后的留存方式。

当重复工作依赖于不应每次重新发现的经验、决策、约束或状态时，请使用记忆模式。

## 当前模式

目前尚无专用的记忆模式。相关的现有模式：

- [持久会话状态 (Durable Session State)](../../../../patterns/orchestration/durable-session-state.md)
- [上下文包 (Context Packet)](../../../../patterns/context/context-packet.md)

- Retrieval Budget。
