# 评估模式

Evaluation pattern 定义人类和代理如何判断某个模式、计划、工作流或输出是否恰当且有效。

当核心问题不是"代理应该做什么？"而是"如何知道这个方法合适且有效？"时，请使用评估模式。

## 当前模式

目前没有公开的评估模式。当足够通用的评估模式通过审查后，将添加到此列表。

## 模式构想

- Evidence Strength。
- Pattern Maturity。
- Counterexample Review。
- Outcome Trace。
