# 提示模式

Prompt pattern 定义可复用的 instruction structure，用于角色、任务、约束和预期输出。

当主要问题不是缺少代码或工具，而是代理行为模糊时，请使用提示模式——即明确代理应优化什么、可以做哪些决策以及必须返回什么。

## 当前模式

- [角色契约 (Role Contract)](../../../../patterns/prompt/role-contract.md)：定义代理角色的 authority、responsibility、boundary、output 和 escalation rule。

## 模式构想

- Reviewer Gate。
- Output Contract。
- Assumption Ledger。
- Decision Escalation。
