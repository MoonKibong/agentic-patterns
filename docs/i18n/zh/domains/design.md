# 设计生成

设计生成涵盖产品设计、界面概念、design-to-code、视觉系统、布局探索和资产移交。

## 常见压力

- 设计输出必须在美学、可用性、品牌、约束和实现成本之间取得平衡。
- 代理可能生成视觉精美但无法实现或维护的制品。
- Design-to-code 需要在视觉意图和技术执行之间进行严格移交。
- 在代价高昂的下游实现启动前，应先进行 human review。
