# 研究与分析

研究和分析工作流需要上下文、来源管控、引用规范、critique 以及人类对结论的判断。

## 常见压力

- 代理可能流畅地生成缺乏支撑的论断。
- 长期研究任务需要持久的来源列表和决策记录。
- Tool output 可能不可信或已过时。
- 人类需要透明的来源溯源，而非仅仅是精练的摘要。
