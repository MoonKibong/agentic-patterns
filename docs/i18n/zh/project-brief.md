# 项目概述

## 工作标题

**Agentic Patterns: A Pattern Language for Human-Guided AI Work**

目前最适合的公开标题。`Agentic Patterns` 这个简洁易记的名字得以保留，同时也阐明了更深远的目标。

## 备选标题

- **Agentic Patterns: Reusable Context for Human-Guided AI**
- **Agentic Patterns: A Catalog of Prompts, Context, and Harnesses**
- **The Agentic Pattern Language**
- **Agentic Patterns: Reusable Knowledge for AI-Assisted Work**

第一个标题最强。本项目不只是 prompt 的集合，而是关于人类用来指挥 AI 代理的语言、上下文、工具和协议。

## 一句话描述

Agentic Patterns 是一个公开目录与模式语言，收录可复用的 prompt、context structure、harness、role contract 和 protocol，帮助人类在重复性的创意、技术和知识工作中更好地引导 AI 代理。

## 简短描述

Agentic Patterns 收集 human-guided AI 工作中反复出现问题的可复用解法。Agentic coding 是第一个成熟的领域，因为它具有明确的上下文文件、工具、测试、仓库、审查循环和持久化制品。同样的模式语言也适用于图像生成、写作、产品设计、design-to-code、video/media production、research 和 analysis。

## 为何重要

AI 代理能够匹配大量模式，但仍然需要人类的目标、判断、记忆和抽象能力。人类往往在能够正式证明某个工作流有效之前，就已经发现了它。为这些工作流命名，使其更易于传达、测试、改进和复用。

本项目将模式视为运营知识的 memetic unit：紧凑、可传播的解法，帮助人类与 AI 系统围绕共同目标协作。

知识背景请参见[模式语言的根源](foundations/pattern-language-roots.md)。

## 项目范围

包含：

- Prompt 和 instruction pattern。
- Context packet 和 memory pattern。
- Planning、execution、review、verification harness。
- Multi-agent orchestration pattern。
- Tool-use pattern。
- Pattern schema 和 agent-readable index。
- Coding、creative generation、design、writing、research 等重复性 human-AI workflow 的 field note 和 evidence。

初期阶段不包含：

- 完整的 hosted pattern registry。
- 声称对所有模式进行普适排名的 benchmark。
- Vendor-specific marketing content。
- 未经整理的通用 AI 新闻或 prompt 集合。

## 初始读者

- 从事 agentic coding 的开发者。
- 使用生成式或 agentic AI workflow 的 designer、writer、researcher、artist、product builder、media creator。
- 编写 Claude Code、Codex、Gemini 等代理指令的人。
- 记录本地 agent workflow practice 的 maintainer。
- 研究 human-AI software work 的实践者和研究者。

## Seed-stage 成功标准

- 仓库说明清晰，足以公开发布到 GitHub。
- 至少三个模式达到期望的质量标准。
- 新贡献者无需借助隐藏上下文即可添加模式。
- 代理能够读取 `AGENTS.md` 并一致地维护目录。
