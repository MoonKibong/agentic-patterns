# 许可协议

本项目采用分离式许可协议。

## 当前决定

采用双许可模式：

- Pattern catalog、文档、prompt、field note：**Creative Commons Attribution 4.0 International (CC BY 4.0)**。
- Code、schema 及已批准的 software artifact：**MIT**。

## 原因

目录是知识成果。CC BY 4.0 在保留署名的同时，鼓励复用、改编和翻译。

Software artifact 使用 MIT，因为它简洁、广为人知，且贡献者易于理解。

## 替代方案

若需要明确的 patent 条款，可对 software artifact 使用 Apache-2.0。

## Repository Files

- `LICENSE`：分离式许可摘要与贡献条款。
- `LICENSE-MIT`：适用于 software artifact 的 MIT license 文本。
- CC BY 4.0 官方地址：https://creativecommons.org/licenses/by/4.0/

公开发布前，请确保贡献者了解其提交的 pattern 和 field note 将在仓库许可协议下被接受。
