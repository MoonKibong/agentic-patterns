# 메모리 패턴

Memory pattern은 유용한 지식이 단일 agent turn, chat session, workflow 이후에도 남는 방식을
정의합니다.

반복 작업이 lesson, decision, constraint, state에 의존하고 매번 재발견하면 안 될 때 사용합니다.
