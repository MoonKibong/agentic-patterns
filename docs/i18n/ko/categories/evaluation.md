# 평가 패턴

Evaluation pattern은 pattern, plan, workflow, output이 적절하고 효과적인지 판단하는 방법을
정의합니다.

중심 질문이 "에이전트가 무엇을 해야 하는가?"가 아니라 "이 접근이 맞고 작동했는지 어떻게
아는가?"일 때 사용합니다.

## 현재 패턴

현재 공개 평가 패턴은 없습니다. 공개하기에 충분히 일반적인 평가 패턴이 검토되면 이
목록에 추가합니다.

## 패턴 아이디어

- Evidence Strength.
- Pattern Maturity.
- Counterexample Review.
- Outcome Trace.
