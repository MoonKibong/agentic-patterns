# Discoverability Plan

이 문서는 Agentic Patterns를 공개한 뒤 사람들이 찾을 수 있게 하기 위한 maintainer checklist입니다.

## Current Positioning

Agentic Patterns는 human-guided AI work를 위한 pattern language입니다. Prompt dump도 아니고,
vendor-specific agent framework도 아닙니다. Coding과 creative workflow 전반에서 context,
prompt, harness, orchestration, tool, memory, evaluation pattern을 이름 붙입니다.
