# Agentic Patterns 위키

Agentic Patterns의 한국어 위키 진입점입니다.

질문에서 시작해 카테고리, 도메인, 개별 패턴으로 이동할 수 있도록 구성되어 있습니다.
영문 문서가 최종 기준이며, 이 문서는 현재 공개 문서의 한국어 미러입니다.

## 시작하기

- [프로젝트 개요](project-brief.md): 프로젝트의 목적과 배경.
- [패턴 언어의 뿌리](foundations/pattern-language-roots.md): Christopher Alexander, GoF, 그리고 agentic patterns의 동기.
- [패턴 언어](pattern-language.md): 패턴 문서의 구조와 품질 기준.
- [증거 위생](evidence-hygiene.md): 비공개 솔루션을 노출하지 않는 예시 작성법.
- [Agentic Work 동향](research/2026-agentic-work-trends.md): 왜 코딩을 넘어 확장하는지.
- [카탈로그](../../../catalog/README.md): 현재 패턴 목록.
- [로드맵](roadmap.md): 앞으로의 방향.
- [기여 가이드](CONTRIBUTING.md): 패턴과 필드 노트 추가 방법.
- [현지화된 빠른 시작](../README.md): 처음 읽는 사람을 위한 짧은 번역.
- [오픈소스 준비](oss/OPEN_SOURCE_READINESS.md): 공개 릴리스 체크리스트.

## 에이전트를 위한 안내

- 수정 전 저장소 루트의 `AGENTS.md`를 읽습니다.
- 공개/비공개 경계를 지키고, 비공개 패턴 이름이나 로컬 도구 이름을 공개 문서에 추가하지 않습니다.
- 패턴 파일을 바꾼 뒤 저장소 검증 절차를 실행합니다.

## 기여자를 위한 안내

- [기여 가이드](CONTRIBUTING.md)에서 시작합니다.
- 새 패턴은 [패턴 템플릿](../../../patterns/_template.md)을 사용합니다.
- 초기 관찰은 [필드 노트 템플릿](../../../field-notes/_template.md)을 사용합니다.
- 공개 릴리스 전 [라이선스](licensing.md)를 확인합니다.
- 예시나 필드 노트를 추가하기 전에 [증거 위생](evidence-hygiene.md)을 확인합니다.
- 사용자 대상 문서를 변경할 때 [현지화된 빠른 시작](../README.md)을 확인합니다.
