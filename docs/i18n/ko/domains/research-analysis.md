# 조사와 분석

Research와 analysis workflow는 context, source control, citation discipline, critique, human
judgment가 필요합니다.

## 공통 압력

- Agent는 근거 없는 claim을 fluently 생성할 수 있습니다.
- Long research task는 durable source list와 decision record가 필요합니다.
- Tool output은 untrusted이거나 stale일 수 있습니다.
- 사람은 polished summary보다 transparent provenance가 필요합니다.
