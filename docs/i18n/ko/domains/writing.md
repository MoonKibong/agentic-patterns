# 글쓰기와 문학 생성

Writing workflow는 brief, outline, draft, critique, revision, continuity, citation, editorial
voice가 필요할 때 agentic pattern을 사용합니다.

## 공통 압력

- Single-pass writing은 polished하게 보이지만 intent를 놓칠 수 있습니다.
- Long-form work는 character, claim, structure, voice를 위한 memory가 필요합니다.
- Critique는 explicit rubric에 기반할 때만 유용합니다.
- Human taste, originality, ethical constraint가 중심입니다.
