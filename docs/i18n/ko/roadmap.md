# Roadmap

이 로드맵은 seed-stage 프로젝트를 집중시키기 위한 문서입니다. 더 좋은 evidence와 pattern
name이 들어오면 바뀔 수 있습니다.

## Phase 0: 언어의 씨앗 만들기

- 프로젝트 title, mission, scope 정의.
- Pattern template과 quality bar 확립.
- 여러 category에 걸친 starter catalog 추가.
- Entry drift를 막기 위한 lightweight validation 추가.

## Phase 1: Field Notes And Evidence

- 관찰된 agentic workflow incident를 위한 `field-notes/` 영역 확장.
- Pattern과 field note 연결.
- Observed pattern과 speculative pattern 구분.
- `draft`에서 `candidate`로 이동하기 위한 maturity rubric 추가.

## Phase 2: Agent-Readable Documentation

- 에이전트용 public pattern index 생성.
- 에이전트가 공개 pattern document를 읽고 인용하는 예시 추가.
- 공개 packaging은 documentation, navigation, sanitized example에 집중.

## Phase 3: Evaluation

- Pattern usefulness를 판단하는 practical metric 정의.
- 반복 task class에서 pattern 비교.
- Counterexample과 deprecated pattern 기록.
- Pattern entry accept/revise review workflow 구축.

## Phase 4: Public Community

- Public release 준비.
- Contribution governance 추가.
- 첫 public release 게시.
- Pattern proposal과 sanitized field note 초대.

## Near-Term Backlog

- Pattern composition example 추가.
- Agent가 planning과 review 중 pattern document를 사용하는 공개 예시 추가.
