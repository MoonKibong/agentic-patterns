# Licensing

이 프로젝트는 split license를 사용합니다.

## 현재 결정

- Pattern catalog, documentation, prompt, field note: **Creative Commons Attribution 4.0 International (CC BY 4.0)**.
- Code, schema, 승인된 software artifact: **MIT**.

## 이유

카탈로그는 knowledge work입니다. CC BY 4.0은 attribution을 유지하면서 reuse, adaptation,
translation을 장려합니다.

Software artifact에는 MIT를 사용합니다. MIT는 단순하고 널리 이해되며 contributor가 이해하기 쉽습니다.

## 대안

명시적인 patent language가 중요하다면 software artifact에 Apache-2.0을 사용할 수 있습니다.

## Repository Files

- `LICENSE`: split-license summary와 contribution terms.
- `LICENSE-MIT`: software artifact에 적용되는 MIT license text.
- CC BY 4.0 공식 URL: https://creativecommons.org/licenses/by/4.0/

공개 전, 기여자가 제출한 pattern과 field note가 repository license 아래 받아들여진다는
점을 이해해야 합니다.
