# Agentic Patterns Wiki

This is the human-readable navigation home for Agentic Patterns.

Use it like a wiki: start from the question you have, follow the category pages, then open the
individual pattern entries when you need operational detail.

## Start Here

- [Project Brief](project-brief.md): what the project is and why it exists.
- [Pattern Language Roots](foundations/pattern-language-roots.md): Christopher Alexander, GoF, and
  the motivation for agentic patterns.
- [Pattern Language](pattern-language.md): how a pattern entry is structured.
- [Evidence Hygiene](evidence-hygiene.md): how to avoid leaking private solution details.
- [Agentic Work Trends](research/2026-agentic-work-trends.md): why this extends beyond coding.
- [Catalog](../catalog/README.md): table of current patterns.
- [Roadmap](roadmap.md): where the project is going.
- [Contributing](../CONTRIBUTING.md): how to add patterns and field notes.
- [Localized Quick Starts](i18n/README.md): short translations for first-time readers.
- [Open Source Readiness](oss/OPEN_SOURCE_READINESS.md): public-release checklist.
