# Code of Conduct

Agentic Patterns aims to be a practical, respectful project for people building better human-guided
AI workflows.

Contributors and maintainers are expected to:

- Use welcoming, professional language.
- Focus criticism on patterns, evidence, examples, documentation, and behavior.
- Assume good intent while still asking for clear evidence.
- Avoid harassment, insults, discriminatory language, and personal attacks.
- Respect maintainer decisions about project scope, evidence hygiene, privacy, and safety.

Unacceptable behavior can be reported privately to the maintainer. Maintainers may remove comments,
close issues, or block participants when needed to keep the project productive.
